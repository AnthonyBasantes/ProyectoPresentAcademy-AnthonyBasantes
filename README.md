# Proyecto Present Academy - Educación de Calidad

## Descripción

Present Academy es una aplicación web interactiva desarrollada como parte del proyecto académico basado en el Objetivo de Desarrollo Sostenible 4 (ODS 4): Educación de Calidad. El sistema tiene como finalidad proporcionar contenido educativo mediante recursos multimedia e interacción entre usuarios, fomentando el aprendizaje de forma dinámica y accesible.

## Tecnologías Utilizadas

* HTML5
* CSS3
* Bootstrap
* JSP (Java Server Pages)
* Java
* Apache Tomcat
* PostgreSQL
* Git y GitHub

## Funcionalidades Principales

### Perfil Invitado

* Visualización de información educativa.
* Consulta del registro de usuarios.

### Perfil Estudiante

* Inicio de sesión en el sistema.
* Interacción con los contenidos educativos.
* Acceso a recursos multimedia.
* Registro de actividades en la bitácora.

### Perfil Administrador

* Gestión de usuarios.
* Creación de nuevos usuarios.
* Actualización de información.
* Consulta de usuarios registrados.
* Bloqueo y desbloqueo de usuarios.
* Consulta de la bitácora del sistema.
* Supervisión de la actividad realizada por los estudiantes.

## Recursos Multimedia Implementados

* Imágenes educativas
* Videos
* Audios
* Modelos 3D
* Elementos de realidad aumentada
* Contenido interactivo

## Seguridad y Validaciones

* Registro mediante correo electrónico válido.
* Contraseñas con longitud mínima de 8 caracteres.
* Control de acceso mediante sesiones.
* Restricción de acceso para usuarios bloqueados.

## Accesibilidad

El proyecto incorpora elementos de accesibilidad para mejorar la experiencia de todos los usuarios, incluyendo:

* Texto alternativo en imágenes.
* Etiquetas descriptivas en formularios.

Además, se realizaron pruebas de accesibilidad utilizando herramientas de validación web.

## Base de Datos

El sistema utiliza PostgreSQL para el almacenamiento de:

* Usuarios.
* Roles.
* Bitácora de actividades.
* Información educativa.
